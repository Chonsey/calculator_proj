# Необходимые команды для работы с окружением

## Создание окружения

`python -m venv venv` 

## Активация окружения

- `venv\Scripts\activate`

- `venv\Scripts\activate.bat`

## Запуск Python script

`python <file_name.py>`

## Деактивация окружения 

`deactivate`
